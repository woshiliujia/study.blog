安装typescipt
```
//全局安装
$ npm install -g typescript

//命令行运行 tsc '文件名'.ts
```
技巧：
多个ts文件一同转换为js
```
$ tsc --init //此命令生产tsconfig.json
$ tsc //转换所有当前文件夹下的ts文件
```
